    Bitmex: Creating API Keys and connecting to QQL

    1. Log in to your account at Bitmex

    2. Follow the link to your Account API Keys Page Bitmex API Management

    3. Select API Parameters

      Start creating your API key by naming it, (e.g. QQL). This name will be shown among the other API keys in your BitMEX account. The CIDR field gives you an ability to restrict your API keys to a single IP address. DO NOT do that. Key Permissions: set them to “Order” and, thus, allow the placement and cancelation of orders via API Key.

    4. Create API Key: Insert the 2FA token if you have 2FA turned on for your account and then press the blue button “Create API key”.

      Congratulations! You have now created your BitMEX API key. You can see both your public key (ID) and your secret key (Secret) on the screen in the yellow box and in the box “Your API keys”. !!! DO NOT CLOSE THIS SCREEN until you add your API keys to QQL – your secret key will only be shown to you this once – if you leave this screen, you’ll never see it again.

    5. Open the QQL API Form page: QQL API Form and log in. At the first usage you need to "Sign up for a new account" and follow the steps.

    6. Enter your name (or alias), select the correct exchange and copy the API Key & API Secret from step 6. Finally press "Submit".

    7. You are all Set and Ready to make Profits with Us!
